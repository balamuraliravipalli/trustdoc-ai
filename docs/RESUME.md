# Resume Positioning

Project title:

**TrustDoc AI — Evidence-Grounded RAG Knowledge Assistant**

Best resume bullet:

> Built a full-stack evidence-grounded RAG assistant using FastAPI, React, OpenAI API, and Qdrant to answer questions over uploaded PDFs with page-level citations, document-type filtering, hallucination guardrails, document management, analytics, and evaluation metrics.

Interview explanation:

TrustDoc AI is an enterprise-style knowledge assistant. It lets admins upload policy/SOP PDFs, chunks and embeds them, stores vectors in Qdrant, retrieves relevant chunks using hybrid retrieval, and uses an LLM to answer only from retrieved evidence. The system shows citations, refuses unsupported questions, logs latency/confidence, and includes a regression evaluation suite.
